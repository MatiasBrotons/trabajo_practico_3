Java

Es una tecnología que incluye un lenguaje de programación y una plataforma de desarrollo.

Una plataforma es el ambiente hardware o software sobre el cual se puede ejecutar un programa. La mayoría de las plataformas actuales se pueden describir como una combinación de un sistema operativo y su hardware subyacente; por ejemplo, un pc con su sistema operativo Windowsse denomina comúnmente plataforma Windows, una notebook con una distribución Linux se denomina plataforma Linux y un celular con Android se conoce como plataforma Android.

La plataforma Java difiere respecto de las plataformas mencionadas en el párrafo anterior debido a que es una plataforma software que se ejecuta sobre las prestaciones que brinda otra plataforma que gestiona el hardware (denominada plataforma base)

Curiosidades de Java

  1. La mascota de Java: Java tiene una mascota oficial conocida como "Duke". Duke es un pingüino de peluche que se ha convertido en un símbolo reconocible para la comunidad de desarrolladores de Java.
  2. Origen del nombre "Java": Como mencioné anteriormente, el nombre "Java" fue elegido después de que la primera elección, "Oak", no estuviera disponible. El nombre Java se inspiró en el café Java, que era popular entre los desarrolladores de Sun Microsystems en ese momento.
  3. Los primeros programas ejecutados en Java: El primer programa ejecutado en Java fue un applet llamado "Hello World". Este applet simplemente mostraba el texto "Hello World" en un navegador web, y se convirtió en un ejercicio común para que los desarrolladores aprendieran los conceptos básicos de Java y la programación orientada a objetos.

Metodos de Java

charAt() Equals compareTo()
Para obtener el carácter situado en una posición determinada de una cadena de caracteres, se debe utilizar el método charAt() Las cadenas de caracteres son de tipo objeto. Hay que utilizar los métodos de la clase String para efectuar comparaciones de cadenas de caracteres. Permite realizar una clasificación entre cadenas.
